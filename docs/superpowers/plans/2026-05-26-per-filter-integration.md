# Per-filter integration breakdown — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Let users record per-filter integration (`{filter, sub_count, sub_exposure_s}`) on a photo — manually, or auto-filled by parsing dropped XISF master headers entirely in-browser — and display the breakdown on the public photo page.

**Architecture:** A JSONB column `photos.filter_integrations` holds an ordered list (Option A from the spec — decoupled from filter chips so luminance is representable). The GET-by-id handler fills it via a dedicated query (mirroring how `tags`/`filter_items` are filled after `From<PhotoRow>`), so the shared `PhotoRow` SELECTs are untouched. The metadata `PUT` accepts it via the existing `double_option` pattern. The frontend adds a verify-form sub-component; a client-side XISF header parser auto-fills rows without uploading the master.

**Tech Stack:** Rust (axum, sqlx `query!` macros, ts-rs), SvelteKit (Svelte 5 runes), vitest.

Spec: `docs/superpowers/specs/2026-05-26-per-filter-integration-design.md`.

```ts
import type { FilterIntegration } from '$lib/api/FilterIntegration';

export function rowTotalS(r: FilterIntegration): number {
  return (Number(r.sub_count) || 0) * (Number(r.sub_exposure_s) || 0);
}
export function grandTotalS(rows: FilterIntegration[]): number {
  return rows.reduce((acc, r) => acc + rowTotalS(r), 0);
}
export function formatHm(totalS: number): string {
  const h = Math.floor(totalS / 3600);
  const m = Math.round((totalS - h * 3600) / 60);
  return `${h} h ${String(m).padStart(2, '0')} m`;
}
```

- [ ] **Step 3: Implement** (mirrors `backend/src/photos/xisf_display.rs`; reads only the header slice, never the body):

```ts
export interface XisfHeaderFacts {
  filter: string | null;
  frames: number | null;
  totalExposureS: number | null;
}

const SIG = 'XISF0100';

export async function parseXisfHeader(file: File): Promise<XisfHeaderFacts | null> {
  const head = new Uint8Array(await file.slice(0, 262_144).arrayBuffer());
  if (head.length < 16) return null;
  if (String.fromCharCode(...head.slice(0, 8)) !== SIG) return null;
  const headerLen = new DataView(head.buffer).getUint32(8, true);
  const end = 16 + headerLen;
  const bytes = end <= head.length ? head : new Uint8Array(await file.slice(0, end).arrayBuffer());
  const xml = new TextDecoder().decode(bytes.slice(16, end));
  const doc = new DOMParser().parseFromString(xml, 'application/xml');
  if (doc.querySelector('parsererror')) return null;

  const fits = (name: string) => {
    for (const k of doc.querySelectorAll('FITSKeyword')) {
      if (k.getAttribute('name')?.toUpperCase() === name) return k.getAttribute('value') ?? null;
    }
    return null;
  };
  const pcl = (id: string) => {
    for (const p of doc.querySelectorAll('Property')) {
      if (p.getAttribute('id') === id) return p.getAttribute('value') ?? p.textContent ?? null;
    }
    return null;
  };
  const unquote = (s: string | null) => s?.trim().replace(/^'|'$/g, '').trim() ?? null;
  const intOf = (s: string | null) => { const n = parseInt((s ?? '').trim(), 10); return Number.isFinite(n) ? n : null; };

  return {
    filter: unquote(fits('FILTER') ?? pcl('Instrument:Filter:Name')),
    frames: intOf(fits('NCOMBINE') ?? pcl('Process:Integration:ImageCount')),
    totalExposureS: decodeF64VecSum(pcl('PCL:TotalExposureTime')),
  };
}

function decodeF64VecSum(b64: string | null): number | null {
  const raw = (b64 ?? '').trim();
  if (!raw) return null;
  let bin: Uint8Array;
  try { bin = Uint8Array.from(atob(raw), (c) => c.charCodeAt(0)); } catch { return null; }
  if (bin.length === 0 || bin.length % 8 !== 0) return null;
  const dv = new DataView(bin.buffer);
  let sum = 0;
  for (let i = 0; i < bin.length; i += 8) sum += dv.getFloat64(i, true);
  return Number.isFinite(sum) && sum > 0 ? sum : null;
}
```
