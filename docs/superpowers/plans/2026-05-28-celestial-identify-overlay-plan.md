# Celestial Identification + Overlay — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Spec:** [`docs/superpowers/specs/2026-05-28-celestial-identify-overlay-design.md`](../specs/2026-05-28-celestial-identify-overlay-design.md)

**Goal:** Auto-identify celestial objects in every plate-solved photo, persist them in `photo_targets` with `source='plate_solve'`, and render a color-coded overlay + side panel on the public photo page; add PGC galaxies to the catalog.

**Architecture:** Three flows (PGC seed → `targets`; platesolve worker → `celestial::identify` in the same tx as `save_result` → `photo_targets`; SSR `load` → `GET /celestial-objects` → `CelestialOverlay` + `CelestialPanel`). Pure SQL bounding-box cone search backed by a B-tree index on `(declination, right_ascension)`. Pure TS gnomonic projection on the client. Aladin Lite reused as-is for the detail subpanel.

**Tech Stack:** Rust (axum + sqlx + serde + anyhow for binaries), PostgreSQL (no extensions), Svelte 5 runes + SvelteKit SSR, vitest (frontend), testcontainers (backend integration), chrome-devtools MCP (E2E).

**PRs:** This plan delivers three independent, mergeable chunks. Each phase is its own PR boundary; the file headers below tag every task with its phase. The phases ship in order (data → backend → frontend) because each consumes the previous one's output.

```rust
fn extract_existing_slug_ref(objname: &str) -> Option<String> {
    let trimmed = objname.trim();
    if trimmed.is_empty() {
        return None;
    }
    // Accept "NGC0224", "NGC 224", "IC0010", "IC 10". Reject subcomponents
    // like "NGC0224A" / "NGC0224-1" — our slug scheme has no equivalent.
    for prefix in ["NGC", "IC"] {
        if let Some(rest) = trimmed.strip_prefix(prefix) {
            let rest = rest.trim_start();
            let digits: String = rest.chars().take_while(|c| c.is_ascii_digit()).collect();
            if digits.is_empty() {
                continue;
            }
            let after_digits = &rest[digits.len()..];
            if !after_digits.is_empty() && after_digits.chars().next() != Some(' ') {
                return None;
            }
            if let Ok(n) = digits.parse::<u32>() {
                return Some(format!("{}-{}", prefix.to_ascii_lowercase(), n));
            }
        }
    }
    None
}
```

```rust
async fn upsert_pgc_row(
    pool: &sqlx::PgPool,
    row: &PgcRow,
) -> Result<UpsertOutcome> {
    let slug = format!("pgc-{}", row.pgc);

    // Dedup: if the objname references an existing NGC/IC slug, skip.
    if let Some(existing) = row.objname.as_deref().and_then(extract_existing_slug_ref) {
        let hit: Option<i64> = sqlx::query_scalar(
            "select 1::int8 from targets where slug = $1",
        )
        .bind(&existing)
        .fetch_optional(pool)
        .await?;
        if hit.is_some() {
            return Ok(UpsertOutcome::DedupedWithNgc);
        }
    }

    let canonical = row
        .objname
        .clone()
        .unwrap_or_else(|| format!("PGC {}", row.pgc));

    // UPSERT: never overwrite canonical_name on UPDATE; always refresh
    // astro fields. Mirrors the OpenNGC seed contract.
    let inserted: bool = sqlx::query_scalar(
        r#"
        insert into targets (
            slug, canonical_name, kind,
            right_ascension, declination, magnitude_v,
            object_type, major_axis_arcmin, minor_axis_arcmin,
            position_angle_deg, updated_at
        ) values ($1, $2, 'pgc', $3, $4, $5, 'G', $6, $7, $8, now())
        on conflict (slug) do update set
            right_ascension     = excluded.right_ascension,
            declination         = excluded.declination,
            magnitude_v         = excluded.magnitude_v,
            object_type         = excluded.object_type,
            major_axis_arcmin   = excluded.major_axis_arcmin,
            minor_axis_arcmin   = excluded.minor_axis_arcmin,
            position_angle_deg  = excluded.position_angle_deg,
            updated_at          = now()
        returning (xmax = 0)
        "#,
    )
    .bind(&slug)
    .bind(&canonical)
    .bind(row.ra_deg)
    .bind(row.de_deg)
    .bind(row.mag_b)
    .bind(row.major_axis_arcmin)
    .bind(row.minor_axis_arcmin)
    .bind(row.position_angle_deg)
    .fetch_one(pool)
    .await?;

    Ok(if inserted { UpsertOutcome::Inserted } else { UpsertOutcome::Updated })
}

#[derive(Default, Debug)]
struct Counts {
    inserted: usize,
    updated: usize,
    deduped_with_ngc: usize,
    skipped_parse: usize,
}

enum UpsertOutcome {
    Inserted,
    Updated,
    DedupedWithNgc,
}
```

```rust
//! Identification entry point. Called from platesolve::save_result.
//! Spec §3.

use crate::celestial::confidence::confidence;
use crate::celestial::queries::{arc_distance_deg, cone_search};
use crate::error::AppError;
use sqlx::{Postgres, Transaction};
use uuid::Uuid;

#[derive(Debug, Default, serde::Serialize)]
pub struct IdentifyOutcome {
    pub found: usize,
    pub kept: usize,
    pub dropped: usize,
}

pub async fn identify(
    photo_id: Uuid,
    tx: &mut Transaction<'_, Postgres>,
) -> Result<IdentifyOutcome, AppError> {
    // 1. Read solve telemetry + image size.
    let row = sqlx::query!(
        r#"select ra_deg, dec_deg, platesolve_pixel_scale_arcsec as scale,
                   width, height
            from photos
           where id = $1"#,
        photo_id,
    )
    .fetch_one(&mut **tx)
    .await?;

    let (ra, dec, scale, w, h) = match (row.ra_deg, row.dec_deg, row.scale, row.width, row.height) {
        (Some(a), Some(d), Some(s), Some(w), Some(h)) if s > 0.0 && w > 0 && h > 0 => {
            (a, d, s, w as f64, h as f64)
        }
        _ => return Ok(IdentifyOutcome::default()),
    };

    // 2. Compute search radius.
    let fov_x_deg = w * scale / 3600.0;
    let fov_y_deg = h * scale / 3600.0;
    let half_diag_deg = 0.5 * (fov_x_deg * fov_x_deg + fov_y_deg * fov_y_deg).sqrt();

    // 3. Get a pool reference for cone_search (which takes &PgPool).
    //    We can't pass tx directly because cone_search is read-only and
    //    the existing pool serves; we run it on the underlying pool via
    //    a fresh acquire — see note. For atomicity, the cone_search read
    //    being outside the tx is fine: writes below are inside.
    //
    // Simpler: change cone_search signature to also accept `&mut Tx`.
    // Done in queries.rs (see Step 2 below if not already done).
    let candidates = crate::celestial::queries::cone_search_in_tx(
        tx, ra, dec, half_diag_deg,
    ).await?;

    let found = candidates.len();

    // 4. Project + filter (F1) + score.
    struct Kept {
        target_id: Uuid,
        confidence: f32,
        is_primary: bool,
    }
    let mut kept: Vec<Kept> = Vec::with_capacity(found);
    for c in &candidates {
        let dist = arc_distance_deg(ra, dec, c.right_ascension, c.declination);
        let major_arcmin = c.major_axis_arcmin.unwrap_or(0.0) as f64;
        let on_screen_px = (major_arcmin * 60.0) / scale;
        let keep = on_screen_px >= 0.5
            || matches!(c.kind.as_str(), "messier" | "ngc" | "ic" | "caldwell");
        if !keep { continue; }
        let conf = confidence(dist, half_diag_deg, on_screen_px, &c.kind, c.magnitude_v);
        kept.push(Kept { target_id: c.id, confidence: conf, is_primary: false });
    }

    let kept_n = kept.len();
    let dropped = found - kept_n;

    // 5. Replace the photo's plate-solve rows.
    sqlx::query!(
        "delete from photo_targets where photo_id = $1 and source = 'plate_solve'",
        photo_id,
    )
    .execute(&mut **tx)
    .await?;

    for k in &kept {
        sqlx::query!(
            r#"insert into photo_targets (photo_id, target_id, source, confidence, is_primary)
                 values ($1, $2, 'plate_solve', $3, $4)
                 on conflict (photo_id, target_id) do nothing"#,
            photo_id,
            k.target_id,
            k.confidence as f64,
            k.is_primary,
        )
        .execute(&mut **tx)
        .await?;
    }

    Ok(IdentifyOutcome { found, kept: kept_n, dropped })
}
```

```rust
pub struct RecomputeResponse {
    pub found: usize,
    pub kept: usize,
    pub dropped: usize,
}

pub async fn recompute(
    State(state): State<AppState>,
    CurrentUser(viewer): CurrentUser,
    Path(photo_id): Path<Uuid>,
) -> Result<Json<RecomputeResponse>, AppError> {
    let owner_id: Uuid = sqlx::query_scalar(
        "select owner_id from photos where id = $1",
    )
    .bind(photo_id)
    .fetch_optional(&state.pool)
    .await?
    .ok_or_else(|| AppError::NotFound("photo".into()))?;
    if viewer.id != owner_id {
        return Err(AppError::Forbidden);
    }

    let mut tx = state.pool.begin().await?;
    let out = crate::celestial::identify(photo_id, &mut tx).await?;
    tx.commit().await?;
    Ok(Json(RecomputeResponse {
        found: out.found,
        kept: out.kept,
        dropped: out.dropped,
    }))
}
```

```rust
//! One-shot identifier for already-solved photos. Idempotent.
//! Run via `just backfill-celestial-objects [--apply] [--reidentify]`.

use anyhow::Result;
use sqlx::PgPool;
use uuid::Uuid;

#[tokio::main]
async fn main() -> Result<()> {
    tracing_subscriber::fmt::init();

    let apply = std::env::args().any(|a| a == "--apply");
    let reidentify = std::env::args().any(|a| a == "--reidentify");

    let database_url = std::env::var("DATABASE_URL")
        .or_else(|_| std::env::var("APP_DATABASE_URL"))?;
    let pool = PgPool::connect(&database_url).await?;

    let candidates: Vec<Uuid> = sqlx::query_scalar(
        r#"select id from photos
             where ra_deg is not null
               and ( $1::bool
                  or not exists (
                     select 1 from photo_targets pt
                      where pt.photo_id = photos.id
                        and pt.source = 'plate_solve'))"#,
    )
    .bind(reidentify)
    .fetch_all(&pool).await?;

    tracing::info!(total = candidates.len(), apply, reidentify, "starting backfill");

    let mut totals = (0_usize, 0_usize, 0_usize);
    for (i, id) in candidates.iter().enumerate() {
        if apply {
            let mut tx = pool.begin().await?;
            match astrophoto::celestial::identify(*id, &mut tx).await {
                Ok(out) => {
                    tx.commit().await?;
                    totals.0 += out.found;
                    totals.1 += out.kept;
                    totals.2 += out.dropped;
                }
                Err(e) => {
                    tracing::warn!(?id, error = %e, "identify failed; skipping");
                    tx.rollback().await?;
                }
            }
        }
        if (i + 1) % 50 == 0 {
            tracing::info!(processed = i + 1, ?totals, "progress");
        }
    }
    tracing::info!(?totals, apply, "backfill complete");
    if !apply {
        tracing::warn!("DRY RUN — re-run with --apply to actually write");
    }
    Ok(())
}
```
